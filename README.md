## ❓ 프로젝트 소개

- 보드게임이 처음이거나 새로운 게임을 하고 싶을 때, 게임을 검색하거나 추천해주는 서비스입니다.
- 보드게임 카페에서 사용하는 키오스크용 태블릿을 생각해 만들었습니다.
- 보드게임 제목을 검색하거나, 선택한 인원 수와 난이도에 해당하는 보드게임을 알려줍니다.

<br>
